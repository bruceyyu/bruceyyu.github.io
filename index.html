GWalkR: R Package for Interactive Exploratory Data Analysis
🕵️ GWalkR is an interactive Exploratory Data Analysis (EDA) Tool in R. With just single line of code, it can simplify your R data analysis and data visualization workflow by turning your data frame into a Tableau-style UI for visual exploration. Also, by using it in RMarkdown, you can create web page to showcase your insights with editable and explorable.
